# Pardus E-Kilit

Okulunuz veya kurumunuz için Pardus tabanlı, QR ve yönetici kodlu kilit uygulaması.

## Kurulum

1. Gerekli paketleri kur:
    ```
    pip3 install -r requirements.txt
    ```

2. Uygulamayı başlat:
    ```
    python3 main.py
    ```

## Özellikler
- QR kod okuma ve kilit özelliği
- 6 haneli yönetici koduyla silme veya açma
- Zamanlayıcı, modern arayüz
- Kendi logonu ve QR kodunu kullanma
